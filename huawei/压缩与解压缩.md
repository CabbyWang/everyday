tar

```shell
# 压缩
tar cvf test.tar test
# 解压缩
tar xvf test.tar
```

tar.gz

```shell
# 压缩
tar -zcvf test.tar.gz test
# 解压缩
tar -zxvf test.tar.gz
```

tar.xz

```shell
# 解压缩
tar xvJf test.tar.xz
```

zip

```shell
# 压缩
zip test.zip test
# 解压缩
unzip test.zip
```

rar

```shell
# 压缩
rar a test.rar test
# 解压缩
rar x test.rar
```
